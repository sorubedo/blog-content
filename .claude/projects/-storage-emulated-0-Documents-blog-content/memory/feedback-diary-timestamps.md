写日记时必须获取系统精确时间（`date` 命令），不只是日期。每条日记内容前用 `**HH:MM**` 格式标注时间戳。

**Why:** 用户反馈"写日记的时候可以获取系统具体的时间而不只是精确到日"，日记是时间流记录，精确到分钟才有意义。

**How to apply:** 每次用户说"日记"时，先用 `date '+%H:%M'` 获取当前时间，写入内容前加上 `**HH:MM**` 时间戳。同一天追加内容时也要标注对应时间。
